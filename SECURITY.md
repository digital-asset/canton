# Security Policy

## Supported Versions

The last 3.x minor release of Canton is supported.
Canton 2.10 is supported as a LTS release.

## Reporting a Vulnerability

In general: If you suspect that your issue has a security impact, do not use the public issue tracker.

Instead report a vulnerability for the following products through Github private vulnerability reports:
* [Daml](https://github.com/digital-asset/daml/security/advisories/new)
* [Canton Ledger](https://github.com/digital-asset/canton/security/advisories/new)
* [Canton Network and Splice](https://github.com/canton-network/splice/security/advisories/new)

Please follow our responsible disclosure policy: https://www.digitalasset.com/responsible-disclosure

For any other security inquiries please contact: security@digitalasset.com.
