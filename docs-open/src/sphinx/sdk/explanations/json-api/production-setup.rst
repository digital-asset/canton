Production Setup
################

The vast majority of prior documentation focused on ease of testing and on setting up the service to run in a dev environment. From a production perspective, given the wide variety of use cases, there is far less of an established framework for deploying a *JSON Ledger API* server. In this document, we provide some recommendations for production deployments.
