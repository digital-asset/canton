Canton Network documentation snippets
---------------------------------------

These reStructuredText pages are a **test harness** for Canton Network documentation: they hold ``snippet`` directives that the ``SphinxDocumentationGenerator`` integration tests execute to produce exported snippets for the published docs site.

They are **not** intended as end-user narrative documentation in this Sphinx build. Use the table of contents to navigate the harness pages grouped by topic (AppDev, SDKs & tools, global synchronizer sections).
